### Initialization servers ###
	OTD-SERVERS.bat

### Launch server ###
	OTD-SERVERS\{SERVER_NAME}\startup.bat

### Stop server ###
	OTD-STOPALL.bat

## Folder Details ##

- **Bin** Kernel openttd.exe & languages
- **Common** newgrfs, scripts and others
- **Servers** servers.
- **Ctl** Control tools
- **Saves** Game saves
